### 기본
```html
선택자 {
    속성명: 속성값;
}
```
선택자: 요소  
속성명: 스타일  
속성값: 어떻게?

p 태그에 color: red 를 적용
```html
p {
    color: red;
}
```

### 문서에 적용하는방법
인라인: 태그에 직접 기술  
스타일: 스타일시트를 위한 태그 추가  
문서간 연결: 따로 작성 후 연결

#### 인라인
```html
<p style="color: red;">
    내용
</p>
```
#### 스타일
```html
<style>
p { color: red;}
</style>
```
### 링크
```html
<link href="style.css" rel="stylesheet">
```
href - 위치  
rel - 연관관계  
head 안에서 사용할 것
