1. 파일 시스템 탐색 기본 명령어

pwd = print working directory

pwd

/home/vagrant/git/bitcamp-ncp

현재 작업 위치를 알려줌

cd = change directory

cd ..

/home/vagrant/git

디렉토리 이동

ls - list

ls

bitcamp-ncp  bitcamp-ncp2  bitcamp-study  bitcamp-test

현재 디렉토리의 리스트를 보여줌

file

file bitcamp-ncp

bitcamp-ncp: directory

파일의 타입을 알려줌

less

less cp.txt

asd (내용)

파일의 내용을 알려줌

2. 파일과 디렉토리 조작 명령어

cp -copy

cp cp.txt new_cp.txt

ls
cp.txt new_cp.txt

파일이나 디렉토리를 복사

mkdir - make directory

mkdir yummi

ls
yummi cp.txt new_cp.txt

새로운 디렉토리를 생성

rm - remove

rm new_cp.txt

ls
yummi cp.txt

삭제 명령어

mv - move

mv cp.txt yummi

ls
yummi

파일이나 디렉토리를 이동

ln - link

파일링크를 만들때 사용

하드링크

ln cp.txt hardlink
ls

cp.txt  hardlink

심볼링크

cd ..
ln -s cp.txt softlink
ls -al

lrwxrwxrwx. 1 vagrant vagrant   6 Nov 23 01:36 softlink -> cp.txt

3. 명령어를 다루는 명령어

type

type mkdir

mkdir is hashed (/usr/bin/mkdir)

명령 유형에 대한 정보를 표시

which

which cp.txt

/usr/bin/which: no cp.txt in (/usr/local/bin:/usr/bin:/usr/local/sbin:/usr/sbin:/home/vagrant/.local/bin:/home/vagrant/bin)

특정 명령어의 위치를 찾을때 사용

man

man mkdir

NAME 
    mkdir - make directories

명령어의 모든 정보를 볼때 사용

apropos

apropos tas

whatis
