Linux Command

1. 파일 시스템 탐색 기본 Command

pwd = 현재 작업 위치

cd = 디렉토리 이동

ls = 현재 디렉토리의 리스트

file = 파일의 타입을 알려줌

less = 파일의 내용을 알려줌

2. 파일과 디렉토리 조작 Command

cp = 파일이나 디렉토리 복사

mkdir = 디렉토리 생성

rm = 삭제 명령어

mv = 파일이나 디렉토리를 이동

ln = 파일링크를 만들때 사용

3. 명령어를 다루는 Command

type = 명령 유형에 대한 정보를 표시

which = 특정 멍령어의 위치를 찾을때 사용

man = 명령어의 모든 정보를 볼때 사용

apropos = 검색한 내용과 관련있는 명령어를 설명과 함께 보여줌

info = 명령어에 대한 사용법과 옵션들을 보여줌

whatis = 명령어에 대한 사용법을 간단하게만 보여줌

alias = 명령어들을 다른 이름으로 바꿀 수 있는쉘 내부 명령어
