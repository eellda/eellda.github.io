World Wide Web

1. 어느 단체의 누가 어떤 목적으로 웹 기술을 만들었는가?

유럽입자물리연구소(CERN) 라는 곳에서
여러 대학과 연구기관에서 일하는 물리학자들 사이에서

"효율적이고 신속한 자료를 공유받기 위한 목적" 으로
1989년에 "팀 버너스리"에 의해 시작된 프로젝트

이후 1990년 말에 HyperText 라는 프로젝트가 생김
HyperText 를 이용한 기술 중 중요한 두 가지가 탄생
이 기술이 바로 "HTML" 과 "HTTP" 라는 기술
세계 최초의 웹 브라우저는 "팀 버너스리"와 "로버트 카이오"가 공동 개발한 월드와이드웹(WWW)

WWW의 기본 아이디어는
진화하는 컴퓨터, 데이터 네트워크 및 하이퍼텍스트 기술을
강력하고 사용하기 쉬운 글로벌 정보 시스템으로 병합하는 것

2. 웹 기술의 처음 목적과 지금 활용되는 상황이 어떻게 다른가?

웹브라우저는 다중 작성자의 환경(소셜 웹)을 뜻했는데
이후에 나오는 브라우저들은 편집의 역할을 빼고 탐색 기능만 제공하게 되어 본래 의도와 다르게 제공

웹 1.0
오로지 ‘읽기’만 가능했고 클릭 외에는 어떠한 상호 작용도 없는 정적인 웹페이지

웹 2.0
읽기’만 가능했던 이용자들은 이제 직접 컨텐츠를 ‘생산’하고 ‘공유’하며 적극적으로 ‘참여’하기 시작
= 커뮤니케이션 가능

웹 3.0
의미론적인 웹’을 뜻하는데 기계가 인간들이 사용하는 자연어를 이해하고 상황과 맥락에 맞는 개인 맞춤형 정보를 제공
= 시맨틱 웹

시맨틱 기술을 활용해 개인 맞춤형 정보를 제공하고
블록체인 시스템을 통한 탈중앙화와 데이터 암호화에 기반한 ‘개인의 데이터 소유’가 가능해진 새로운 형태의 웹
필요 없는 정보를 제거하고 사용자의 성향과 검색 목적에 따른 개인 맞춤형 정보를 신속하게 도출이 가능해짐
